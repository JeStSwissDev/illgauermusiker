🎵 Musikwelt Illgau
Webanwendung zur Erfassung und Anzeige von Musikern, Formationen, Instrumenten und Genres eines Dorfes — inklusive Mitgliedschaften, Zeiträumen, Fotos, MP3 und QR-Codes.

✨ Funktionen
Musiker- und Formation-Archiv mit Biografien
Instrumente & Genres verwalten
Mitgliedschaften mit Zeitraum (wer spielte wann wo)
Öffentliche Detailseiten per URL oder QR-Code
Geschützter Erfassungsbereich (Login)
Datei-Uploads (Bilder, MP3)

🏗️ Technologie-Stack
Backend
  Node.js + Express
  MySQL / MariaDB (mysql2)
  express-session (Login)
  Multer (Uploads)
  dotenv (Konfiguration)
Frontend
  HTML + CSS
  Vanilla JavaScript
  qrcode.js
